# Mugisha Micheal Portfolio

High-impact, multi-page developer portfolio built with HTML, CSS, vanilla JS, and Bootstrap. Includes project case studies, blog, testimonials, and a persistent light/dark theme toggle.

## Highlights
- Multi-page site with consistent navigation and footer
- Case study pages for each project
- Smooth reveal animations and modern UI styling
- Light/dark mode with saved preference

## Pages
- `src/index.html`
- `src/about.html`
- `src/projects.html`
- `src/testimonials.html`
- `src/blog.html`
- `src/contact.html`
- Case studies: `src/project-*.html`

## Structure
- `src/css/styles.css` global styling and theme variables
- `src/js/main.js` animations, theme toggle, and active nav
- `src/assets/` images, icons, and CV

## Run Locally
Option 1: Open `src/index.html` in a browser.  
Option 2: Serve the `src/` folder with any static server.

## Customize
- Update content directly in the HTML files.
- Replace images in `src/assets/img/`.
- Update the CV at `src/assets/MUGISHA-MICHEAL-CV.pdf`.
- Edit project case studies in `src/project-*.html`.

## Deploy
This is a static site. You can deploy the `src/` folder to:
- Vercel
- Netlify
- GitHub Pages

## Tech
- HTML
- CSS
- Vanilla JS
