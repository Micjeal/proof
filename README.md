# Mugisha Micheal Portfolio

Modern multi-page portfolio built with HTML, CSS, vanilla JS, and Bootstrap. Includes case studies, blog, testimonials, and a light/dark theme toggle.

## Pages
- `src/index.html`
- `src/about.html`
- `src/projects.html`
- `src/testimonials.html`
- `src/blog.html`
- `src/contact.html`
- Case studies: `src/project-*.html`

## Run Locally
Open `src/index.html` in a browser, or serve the `src/` folder with any static server.

## Customize
- Update content in the HTML files.
- Replace images in `src/assets/img/`.
- Update the CV at `src/assets/MUGISHA-MICHEAL-CV.pdf`.

## Tech
- HTML
- CSS
- Vanilla JS
- Bootstrap 5
